## 2024-06-09 - Add focus visible styling to Topbar icon buttons\n**Learning:** Icon buttons used in the top navigation were not receiving keyboard focus styles, making the topbar completely inaccessible via keyboard navigation. Sighted keyboard users could tab through but wouldn't know which button was focused. Missing `focus-visible` utility classes on custom `<button>` components.\n**Action:** Always include `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2 focus-visible:ring-offset-background` on custom interactive elements that do not already have keyboard focus styling.
